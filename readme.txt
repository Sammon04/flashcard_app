Starting db:
clone the repo into c:/xampp/htdocs
Open Xampp and start Apache and MySQL
go to localhost/phpmyadmin
Create database named flashcard_app
import schema.sql first
then import data.sql (disable foregin key checks before running it though)

Starting frontend:
open bash terminal in vscode and go to /frontend
do npm install
do npm run dev
go to the localhost url that it gives you
